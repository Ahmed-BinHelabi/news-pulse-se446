## Team Members

## Overview

This project builds a live news analytics pipeline using:

- RSS ingestion
- PySpark Structured Streaming
- Streamlit dashboard
- LLM-generated summaries

## Architecture

RSS Feeds → JSON Files → Spark Streaming → Memory Tables → Streamlit Dashboard

## How to Run

### 1. Install dependencies

```bash
pip install pyspark==3.5.0 feedparser pandas streamlit requests
```
